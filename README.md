## Getting Started

### Clone the Repository

```bash
git clone https://github.com/KUAS-ubicomp-lab/GlucoPatrol.git
cd your-repo
git checkout dev # Switch to branch 'dev'
```

### Set Up Enviroment

#### Windows

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

#### macOS / Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
